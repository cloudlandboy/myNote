## 离线查看

1. 确保已经安装了 `node` 环境

2. 全局安装 `docsify-cli`

   ```shell
   npm i docsify-cli -g
   ```

3. clone到本地并启动

   ```shell
   git clone https://github.com/cloudlandboy/myNote.git
   cd myNote/
   docsify serve docs/
   ```

4. 打开浏览器并访问 [http://127.0.0.1:3000/](http://127.0.0.1:3000/)
